Chapter 3 - SVG (Scalable Vector Graphics)

<svg viewbox = "0 0 100 100"> // Initialize a box with 0 x-translation 0 y-translation within a 100 width 100 length frame

<path d="M xy l xy C/Q xy, xy, xy"> </svg>

//
d: Prepare viewbox for the intent to draw an object
M: Define the co-ordinates within the viewbox to start drawing
l: Create a straight line
C/Q: Create a curved line
fill= SVG equivalent to background-color
stroke= SVG equivalent to border
